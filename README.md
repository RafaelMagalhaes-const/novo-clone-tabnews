# Aviação News

Plataforma de compartilhamento de conteúdo inspirada no TabNews, desenvolvida como projeto de estudo durante o curso.dev.

## Sobre o projeto

O Aviação News tem como objetivo replicar os principais conceitos de uma aplicação fullstack moderna, incluindo:

- Criação e exibição de posts
- Estruturação de rotas e API
- Integração entre frontend e backend
- Organização de código baseada em boas práticas

## Tecnologias utilizadas

- JavaScript
- Node.js
- Next.js
- React
- Vercel

## Aprendizados

Durante o desenvolvimento, foram praticados conceitos como:

- Estruturação de aplicações web
- Separação de responsabilidades
- Consumo de APIs
- Controle de fluxo e lógica de aplicação

## Observações

Este projeto foi desenvolvido para fins educacionais, inspirado na arquitetura do TabNews.

Este projeto está sendo desenvolvido durante o curso.dev do Filipe Deschamps
